
# AIFIX Enterprise ESG Portal

This is a code bundle for AIFIX Enterprise ESG Portal. The original project is available at https://www.figma.com/design/IVRX9mpA2eVIQGB7NRohRA/AIFIX-Enterprise-ESG-Portal.

## 기술 스택

- Next.js 16 (App Router)
- React 18
- TypeScript
- Tailwind CSS

## 실행 방법

### 의존성 설치

```bash
npm install
# 또는
pnpm install
```

### 개발 서버 실행

```bash
npm run dev
# 또는
pnpm dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

### 프로덕션 빌드

```bash
npm run build
npm start
```

## 프로젝트 구조

- `/app` - Next.js App Router 페이지 및 레이아웃
- `/src/components` - React 컴포넌트
- `/src/ui` - UI 컴포넌트 라이브러리
